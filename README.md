## 🎨 How to Customize

### 1. Add Your Photos
Replace the placeholder images in `index.html`:
- Look for `<div class="photo-card placeholder">`
- Replace with actual `<img>` tags like this:
```html
<div class="photo-card">
  <img src="your-photo.jpg" alt="Couple Photo 1" />
  <p class="caption">Your caption here 💕</p>
</div>
```

### 2. Add Videos
Replace video placeholders in `index.html`:
```html
<div class="video-card">
  <video controls>
    <source src="your-video.mp4" type="video/mp4" />
  </video>
</div>
```

### 3. Add Background Music (Optional)
In `script.js`, uncomment line 20 and add your audio file:
```javascript
bgMusic.src = "your-audio-file.mp3";
```

### 4. Change Colors (Optional)
Edit `style.css` to change the color scheme:
- Main colors: Pink (#ff6b9d), Purple (#c44569), Gold (#ffd700)
